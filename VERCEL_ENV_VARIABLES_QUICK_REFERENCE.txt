üìù SETUP INSTRUCTIONS
================================================================================

1. Go to: https://vercel.com/your-project/settings/environment-variables

2. For each variable above:
   - Click "Add New"
   - Enter the variable name
   - Enter the variable value (replace placeholders!)
   - Select "Production", "Preview", and/or "Development"
   - Click "Save"

3. After adding all variables:
   - Go to Deployments tab
   - Click "Redeploy" on the latest deployment
   - OR trigger a new deployment by pushing to your repository

4. Test your deployment:
   - Visit: https://lifeset.vercel.app/api/v1/docs
   - Try logging in
   - Check Vercel logs for any errors

================================================================================
üö® COMMON FIXES
================================================================================

Issue: "A server error has occurred"
Fix: Ensure DATABASE_URL and JWT_SECRET are set correctly

Issue: "Database connection error"
Fix: Make sure DATABASE_URL doesn't use localhost, use production database URL

Issue: "CORS error"
Fix: Add your frontend domain to CORS_ORIGIN

Issue: "Authentication failed"
Fix: Check that JWT_SECRET and JWT_REFRESH_SECRET are set and different
